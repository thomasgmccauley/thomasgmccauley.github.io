# Thomas G. McCauley Academic Website

This folder is ready to upload to GitHub Pages. The site uses plain HTML and CSS only.

## Files

- `index.html` — home page
- `cv.html` — embedded CV viewer
- `publications.html` — publication list with inline PDF previews
- `contact.html` — contact page
- `style.css` — shared site styling
- `headshot.jpg` — homepage portrait
- `cv.pdf` — CV embedded on the CV page
- `manuscripts/` — publication PDFs embedded on the Publications page
